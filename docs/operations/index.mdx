### Daily Operations

**Morning Health Check (15 min)**
- Review Azure Monitor dashboards
- Check APIM capacity and error rates
- Review overnight alerts
- Verify backup completion

**Log Review (10 min)**
- Check for error patterns
- Review failed authentication attempts
- Monitor policy violations
- Note anomalies for investigation

### Weekly Operations

**Performance Analysis (30 min)**
- Review latency trends (P50, P95, P99)
- Analyze capacity utilization
- Check error rate trends
- Identify optimization opportunities

**Security Review (20 min)**
- Review access logs
- Check for policy violations
- Verify certificate expiry dates
- Review security alerts

### Monthly Operations

**Maintenance Window Activities:**
- Certificate rotation (if needed)
- Access reviews
- Documentation updates
- Capacity planning review

### Quarterly Operations

**Strategic Activities:**
- Disaster recovery testing
- Security audits
- Cost optimization review
- Architecture review

|-----------|-----------|---------------|------|

## Incident Response

### Severity Levels

| Level | Definition | Response Time | Example |
|-------|------------|---------------|---------|
| **P1 - Critical** | Complete outage | 15 min | All APIs down |
| **P2 - High** | Major impairment | 30 min | High error rate |
| **P3 - Medium** | Minor impairment | 2 hours | Degraded performance |
| **P4 - Low** | No user impact | 24 hours | Documentation issue |

### Escalation Path

```
L1: On-call Engineer (15 min)
  └─▶ L2: Platform Team Lead (30 min)
        └─▶ L3: Engineering Manager (1 hour)
              └─▶ L4: Executive Team (2 hours)
```

## Tools and Resources

### Azure CLI Commands

```bash
# Health check
az apim show -n <apim> -g <rg> --query "{state:provisioningState, capacity:sku.capacity}"

# Recent errors
az monitor log-analytics query \
  --workspace <ws> \
  --analytics-query "ApiManagementGatewayLogs | where TimeGenerated > ago(1h) | where responseCode >= 400 | count"

# Capacity metrics
az monitor metrics list \
  --resource <apim-id> \
  --metric "Capacity" \
  --interval PT5M
```

### Log Analytics Queries

```kusto
// Daily health summary
ApiManagementGatewayLogs
| where TimeGenerated > ago(24h)
| summarize 
    TotalRequests = count(),
    ErrorRate = countif(responseCode >= 400) * 100.0 / count(),
    P95Latency = percentile(DurationMs, 95)
    by bin(TimeGenerated, 1h)
| render timechart

// Top error sources
ApiManagementGatewayLogs
| where TimeGenerated > ago(24h)
| where responseCode >= 400
| summarize Count = count() by apiId, responseCode
| order by Count desc
```
