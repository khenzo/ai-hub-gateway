# Guides

This section provides deployment and implementation guidance for two complementary Azure solutions that work together to deliver enterprise-grade AI infrastructure.

## Getting Started

The documentation covers two solutions that are designed to be deployed sequentially:

1. **AI Landing Zone** — Foundational Azure infrastructure for AI workloads, including hub-spoke networking, governance policies, and landing zone architecture.
2. **Citadel Hub** — An Azure API Management-based AI gateway that adds LLM routing, PII masking, throttling, and usage analytics on top of the landing zone.

<Tip>
  **Recommended path:** Start with AI Landing Zone to establish the core Azure infrastructure, then add Citadel Hub for AI gateway capabilities.
</Tip>
