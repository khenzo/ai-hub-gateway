# Citadel Hub

Citadel Hub is an Azure API Management-based AI gateway that adds enterprise capabilities to your AI Landing Zone, including LLM routing, PII masking, throttling, usage analytics, and Entra ID authentication.

## What It Provides

- **LLM Routing** — Intelligent request routing and load balancing across multiple AI backends
- **PII Masking** — Protect sensitive data in AI requests before it reaches external models
- **Throttling & Rate Limiting** — Manage quotas and prevent overload with policy-based controls
- **Usage Analytics** — Monitor consumption, costs, and performance with built-in dashboards
- **Entra ID Integration** — Secure access with Azure Active Directory authentication
