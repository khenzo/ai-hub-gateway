## How to use

<Steps>
  <Step title="Open the deck">
    Click the card above to launch the presentation in a new tab.
  </Step>
  <Step title="Navigate">
    Use **← / →** arrow keys to move between slides. Press **F** to toggle fullscreen.
  </Step>
  <Step title="Present">
    Decks render at 1920×1080 (16:9) — connect to a projector or external display for best results.
  </Step>
</Steps>

<Note>
  Decks are hosted on the same CloudFront distribution as this documentation site. Source files live in the `presentation/` folder of the repository and are deployed via `./deploy-s3.sh`.
</Note>
