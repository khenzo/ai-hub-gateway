# Post-Deployment Validation

Run these validation procedures immediately after deploying AI Landing Zone infrastructure to confirm correct operation before enabling production workloads.

<Note>
Allow 4-8 hours for complete validation. Do not skip steps—each validates critical functionality that production workloads depend on.
</Note>

---

## Smoke Tests

### Resource Deployment Verification

Verify all resources deployed successfully and are in the correct state:

| # | Test | Command | Expected Result | Status |
|---|------|---------|-----------------|--------|
| **1.1** | APIM service state | `az apim show --name <apim-name> -g <rg> --query provisioningState` | `"Succeeded"` | [ ] |
| **1.2** | VNet state | `az network vnet show --name <vnet-name> -g <rg> --query provisioningState` | `"Succeeded"` | [ ] |
| **1.3** | Key Vault state | `az keyvault show --name <kv-name> --query properties.provisioningState` | `"Succeeded"` | [ ] |
| **1.4** | Log Analytics workspace | `az monitor log-analytics workspace show -g <rg> -n <workspace> --query provisioningState` | `"Succeeded"` | [ ] |
| **1.5** | Resource count matches plan | Count resources in resource group | Matches expected count | [ ] |

**Comprehensive resource check:**

```bash
#!/bin/bash
# smoke-test-resources.sh

RG="<resource-group>"
APIM="<apim-name>"

echo "=== Resource Smoke Tests ==="
echo ""

# Check APIM
APIM_STATE=$(az apim show --name "$APIM" -g "$RG" --query "provisioningState" -o tsv 2>/dev/null)
if [ "$APIM_STATE" = "Succeeded" ]; then
    echo "✅ APIM: Succeeded"
else
    echo "❌ APIM: $APIM_STATE"
fi

# Check VNet
VNET_STATE=$(az network vnet list -g "$RG" --query "[0].provisioningState" -o tsv)
echo "VNet State: ${VNET_STATE:-"Not Found"}"

# List all resources with their states
echo ""
echo "All Resources:"
az resource list -g "$RG" --query "[].{name:name, type:type, state:provisioningState}" -o table
```

### Rate Limiting Verification

| # | Test | Method | Expected Result | Status |
|---|------|--------|-----------------|--------|
| **3.8** | Rate limit enforced | Send 100+ requests rapidly | HTTP `429` after threshold | [ ] |
| **3.9** | Retry-after header present | Inspect 429 response | `Retry-After` header present | [ ] |
| **3.10** | Rate limit resets | Wait for window, retry | Request succeeds | [ ] |

**Rate limit test:**

```bash
#!/bin/bash
# rate-limit-test.sh

GATEWAY="https://<apim-name>.azure-api.net"
API_PATH="/echo/resource"
SUBSCRIPTION_KEY="<your-key>"
RATE_LIMIT=100  # Per your contract

echo "Sending $((RATE_LIMIT + 20)) requests..."
for i in $(seq 1 $((RATE_LIMIT + 20))); do
    RESPONSE=$(curl -s -o /dev/null -w "%{http_code}" \
        -H "Ocp-Apim-Subscription-Key: $SUBSCRIPTION_KEY" \
        "$GATEWAY$API_PATH")
    
    if [ "$RESPONSE" = "429" ]; then
        echo "Rate limit hit at request $i"
        break
    fi
done
```

## Performance Baseline

### Latency Tests

Establish baseline latency metrics for your deployment:

| Metric | Target | Critical | Test Command |
|--------|--------|----------|--------------|
| **P50 Response Time** | < 1s | > 3s | See script below |
| **P95 Response Time** | < 3s | > 10s | See script below |
| **P99 Response Time** | < 5s | > 15s | See script below |

**Latency test script:**

```bash
#!/bin/bash
# latency-baseline.sh

GATEWAY="https://<apim-name>.azure-api.net"
API_PATH="/openai/deployments/<model>/chat/completions?api-version=2024-02-01"
SUBSCRIPTION_KEY="<your-key>"
REQUESTS=100

echo "Running $REQUESTS requests for latency baseline..."

# Create temp file for timing data
TIMINGS_FILE=$(mktemp)

for i in $(seq 1 $REQUESTS); do
    TIME=$(curl -s -o /dev/null -w "%{time_total}" \
        -H "Content-Type: application/json" \
        -H "Ocp-Apim-Subscription-Key: $SUBSCRIPTION_KEY" \
        -d '{"messages":[{"role":"user","content":"Hello"}],"max_tokens":10}' \
        "$GATEWAY$API_PATH")
    echo "$TIME" >> "$TIMINGS_FILE"
done

# Calculate statistics
echo ""
echo "=== Latency Baseline Results ==="
echo "Samples: $REQUESTS"
echo "P50: $(sort -n "$TIMINGS_FILE" | awk 'NR==50{print $1*1000"ms"}')"
echo "P95: $(sort -n "$TIMINGS_FILE" | awk 'NR==95{print $1*1000"ms"}')"
echo "P99: $(sort -n "$TIMINGS_FILE" | awk 'NR==99{print $1*1000"ms"}')"

rm "$TIMINGS_FILE"
```

#!/bin/bash

## Validation Scripts

### Complete Post-Deployment Validation

```bash
#!/bin/bash
# post-deployment-validation.sh
# Run all validation tests

set -e

RG="<resource-group>"
APIM="<apim-name>"
VNET="<vnet-name>"
KV="<keyvault-name>"
OPENAI="<openai-name>"

echo "========================================="
echo "Post-Deployment Validation Suite"
echo "========================================="
echo ""

PASSED=0
FAILED=0

run_test() {
    local name="$1"
    local command="$2"
    local expected="$3"
    
    echo -n "Testing: $name... "
    
    if RESULT=$(eval "$command" 2>/dev/null); then
        if [[ "$RESULT" == *"$expected"* ]] || [[ "$RESULT" == "$expected" ]]; then
            echo "✅ PASS"
            ((PASSED++))
        else
            echo "❌ FAIL (got: $RESULT, expected: $expected)"
            ((FAILED++))
        fi
    else
        echo "❌ FAIL (command error)"
        ((FAILED++))
    fi
}

# Smoke Tests
run_test "APIM State" \
    "az apim show -n $APIM -g $RG --query provisioningState -o tsv" \
    "Succeeded"

run_test "VNet State" \
    "az network vnet show -n $VNET -g $RG --query provisioningState -o tsv" \
    "Succeeded"

# Connectivity Tests
GATEWAY_URL=$(az apim show -n "$APIM" -g "$RG" --query gatewayUrl -o tsv)
run_test "Gateway Reachable" \
    "curl -s -o /dev/null -w '%{http_code}' $GATEWAY_URL" \
    "200"

echo ""
echo "========================================="
echo "Results: $PASSED passed, $FAILED failed"
echo "========================================="

if [ $FAILED -gt 0 ]; then
    exit 1
fi
```

## Success Criteria

### All Tests Pass Definition

Deployment is successful when:

- [ ] **100%** of smoke tests pass
- [ ] **100%** of connectivity tests pass
- [ ] **100%** of security tests pass
- [ ] **90%+** of functionality tests pass
- [ ] **100%** of integration tests pass
- [ ] Performance meets **baseline targets**

### Performance Thresholds

| Component | P50 Latency | P95 Latency | Availability |
|-----------|-------------|-------------|--------------|
| APIM Gateway | < 50ms overhead | < 100ms overhead | 99.9% |
| Azure OpenAI | Model-dependent | < 5s | 99.5% |
| End-to-end | < 2s | < 5s | 99.5% |

### Remediation Steps for Failures

| Failure Type | Immediate Action | Escalation |
|--------------|------------------|------------|
| Resource in failed state | Review Activity Log, retry deployment | Platform Engineer |
| Network connectivity issues | Check NSG rules, VNet peering | Network Engineer |
| Authentication failures | Verify APIM policies, Key Vault access | Security Engineer |
| Performance below threshold | Check capacity, scale if needed | SRE |
| Missing logs/alerts | Verify diagnostic settings | DevOps Engineer |
