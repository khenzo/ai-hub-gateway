    AI Landing Zone

    The AI Landing Zone provides the foundational Azure infrastructure for running secure, scalable AI workloads. It establishes the hub-spoke network topology, governance policies, and landing zone architecture required before adding AI gateway capabilities.

    ​
    What It Provides

    • Hub-spoke networking with centralized connectivity and network segmentation
    • Governance and policy foundation via Azure Policy initiatives
    • Landing zone architecture ready for AI workload deployment
    • Secure baseline for generative AI applications at enterprise scale

    ​
    Official Repository

    Azure/AI-Landing-Zones

    View the Bicep modules, deployment scripts, and architecture documentation for the AI Landing Zone.

    ​
    Recommended Path

    Start here before deploying Citadel Hub. The AI Landing Zone creates the Azure foundation that Citadel Hub extends with AI gateway capabilities.
