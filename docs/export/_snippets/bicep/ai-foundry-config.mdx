# AI Foundry Configuration

## Foundry Instance Configuration

```bicep
param aiFoundryInstances = [
  {
    name: 'foundry-primary'
    location: 'eastus'
    customSubDomainName: ''
    defaultProjectName: 'citadel-governance-project'
  }
  {
    name: 'foundry-secondary'
    location: 'westus2'
    customSubDomainName: ''
    defaultProjectName: 'citadel-governance-project'
  }
]
```
