# Quickstart Guide

Deploy your first AI Landing Zone in under 30 minutes.

## Prerequisites

Before you begin, ensure you have:

<Check>Azure subscription with appropriate permissions</Check>
<Check>Azure CLI installed and configured</Check>
<Check>Git access to deployment templates</Check>

## Deployment Steps

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/example/ai-landing-zone.git
    cd ai-landing-zone
    ```
  </Step>
  
  <Step title="Configure Environment">
    Copy the sample configuration and customize for your environment:
    ```bash
    cp config/sample.env config/production.env
    # Edit config/production.env with your settings
    ```
  </Step>
  
  <Step title="Run Deployment">
    Execute the deployment script:
    ```bash
    ./scripts/deploy.sh --environment production
    ```
  </Step>
  
  <Step title="Verify Deployment">
    Check that all resources are provisioned:
    ```bash
    ./scripts/verify.sh --environment production
    ```
  </Step>
</Steps>
