Governed Data Query Path

Every agent data query traverses the full stack:

Step Component Action
1 AI Agent Sends HTTPS request with Entra Bearer Token + APIM key
2 APIM Gateway Validates token, enforces QPM quota, emits usage event
3 Databricks SQL Executes query through Unity Catalog with MI token
4 Unity Catalog Applies row filters and column masks at query time
5 OneLake Returns governed result set with Entra ID auth

Access Contracts & Rate Limiting

Access Contracts bind policies to identities, preventing cost overruns and ensuring fair resource allocation.

Contract Structure

  • Identity (team, user, or agent)
  • Environment (dev/staging/prod)
  • Policy set (rate limits, safety, budgets)
  • Validity period with auto-expiry

Limit Types

  • Request Rate — 100 req/min
  • Token Quota — 1M tokens/day
  • Concurrent — 10 concurrent requests
  • Burst — 200 req/min burst
