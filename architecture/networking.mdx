# Networking Architecture

Secure network design for enterprise AI workloads.

## Network Topology

The AI Landing Zone supports two deployment patterns:

### Hub Network

Deploy within your existing hub VNet for direct connectivity.

<Accordion title="Benefits">
- Simplified routing
- Direct spoke communication
- Shared security policies
</Accordion>

### Spoke Network

Deploy in a dedicated spoke with hub peering for enhanced isolation.

<Accordion title="Benefits">
- Network isolation
- Granular security controls
- Clear boundaries
</Accordion>

## Security Features

- Private endpoints for all services
- Network security groups
- Azure Firewall integration
- DDoS protection

## Connectivity

- VNet peering
- ExpressRoute support
- VPN Gateway ready
- Hybrid connectivity
