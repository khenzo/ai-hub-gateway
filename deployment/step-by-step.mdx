# Step-by-Step Deployment

Complete deployment guide for the AI Landing Zone.

## Deployment Methods

### Azure Developer CLI (Recommended)

```bash
# Authenticate
azd auth login

# Initialize
cd ai-landing-zone
azd init --template Azure-Samples/ai-hub-gateway-solution-accelerator -e my-env --branch citadel-v1

# Deploy
azd up
```

### Manual Deployment

1. Clone the repository
2. Configure parameters
3. Deploy Bicep templates
4. Validate deployment

## Configuration

Key parameters to configure:

| Parameter | Description | Example |
|-----------|-------------|---------|
| `environmentName` | Deployment environment | `production` |
| `location` | Azure region | `eastus` |
| `networkType` | Hub or spoke deployment | `hub` |
