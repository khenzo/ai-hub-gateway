# Deployment Prerequisites

Ensure you have everything needed before starting deployment.

## Azure Requirements

<Check>Azure subscription with Owner or Contributor + User Access Admin</Check>
<Check>Resource group created in target subscription</Check>
<Check>Required resource providers registered</Check>

## Tools

<Check>Azure CLI installed</Check>
<Check>Azure Developer CLI (azd) installed</Check>
<Check>Git access to repository</Check>

## Permissions

Required permissions:
- Subscription Owner, OR
- Subscription Contributor + User Access Administrator

## Network

Decide on your network approach:
- Existing hub VNet
- New spoke VNet
- CIDR ranges for subnets
